# React BeUI

React 전용 UI 컴포넌트 라이브러리

`react-be-ui`는 `React`전용 UI 컴포넌트 라이브러리입니다.
디자인 시스템이 적용된 [`Flog`](https://frog-ui-css.vercel.app/) CSS 라이브러리를 기반으로 최신의 `React`에 사용 가능한 컴포넌트로 이루어저 있습니다.

- 총 23개 컴포넌트 구현
- NPM 패키지 배포
